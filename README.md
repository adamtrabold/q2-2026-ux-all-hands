# Q2 2026 UX All Hands — On Evolution

Presentation slides for the Q2 2026 UX team all hands meeting, focusing on the theme of evolution in the age of AI tools.

## Viewing the Presentation

**Live URL:** https://adamtrabold.github.io/q2-2026-ux-all-hands/

Or download and open `index.html` in a web browser.

**Navigation:**
- Arrow keys or click to advance slides
- Press `?` for keyboard shortcuts (if available)

## Key Themes

- **Skeptic vs. Debunker** framework
- **We can finally cook** — AI as enabler, not replacement
- **Automation vs. Assistance** — matching tasks to mindset
- **Future is for makers** — deep technical understanding drives great products
